# SimpleStorage with Ethers

Based on [PatrickAlphaC/ethers-simple-storage-fcc](https://github.com/PatrickAlphaC/ethers-simple-storage-fcc).

## Instructions

Set up envs:

```sh
mv .env.example .env
```

Compile contracts:

```sh
yarn compile
```

Run script:

```sh
yarn ts-node <script.ts>
```

<br>

Encrypt private key:

```sh
PRIVATE_KEY_PASSWORD= node encryptKey.js
```
